# ASTA API

Sistema de gestión de libros utilizando un LLM para operaciones CRUD y funcionalidades avanzadas.

## Descripción

De momento es un proyecto sobre un CRUD de libros que utiliza un modelo de lenguaje (LLM) para procesar y gestionar la información.

El proyecto incluirá las siguientes herramientas:
- **Búsqueda de libros**: Sistema de búsqueda avanzada
- **Reseñas**: Gestión y análisis de reseñas de libros
- **Recomendaciones**: Motor de recomendaciones personalizadas basado en LLM
